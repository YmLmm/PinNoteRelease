# PinNote 发布仓库

**PinNote**（Se7en 出品）— 桌面便签 / 磁贴应用的官方发布仓库。

## 下载

前往 [**Releases**](https://github.com/YmLmm/PinNoteRelease/releases/latest) 下载最新安装包：

- Windows：`PinNote_x.y.z_x64-setup.exe`
